# Car Price Prediction using Linear Regression

This project uses Linear Regression to predict car prices based on vehicle features.

## Dataset
CarPrices dataset with 205 samples and multiple vehicle attributes.

## Features
- enginesize
- horsepower
- curbweight
- carwidth
- fueltype
- drivewheel

## Target
price

## Model
Linear Regression

## Results
MAE: 2021  
RMSE: 3210
